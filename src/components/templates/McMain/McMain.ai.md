# McMain — для ИИ

## Назначение

Каркас основной страницы: левая колонка (сайдбар), верхняя панель, контент и опциональный футер. Выставляет CSS-переменную **`--vh`** для корректной высоты на мобильных.

## Критичные props

- Пропов почти нет; вёрстка через **слоты**.

## Слоты

- **`left-bar`**: сайдбар.
- **`top-bar`**: шапка (часто `McTopBar`).
- **`default`**: основной контент.
- **`footer`**: подвал страницы.

## Композиция

- Обычно корень layout приложения; внутрь слотов ставят `McSideBar`, `McTopBar`, `router-view`.

## Частые ошибки

- Ожидать пропы маршрутизации — их нет, только разметка и `--vh`.
